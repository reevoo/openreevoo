Openreevoo
==========

Elixir client for writing reviews of elixir libraries.

## Build and run

Build the executable:
```sh
mix escript.build
```

Run the executable:

```sh
./openreevoo
```


## Development

```sh
# Run the tests
mix test

# Run the tests automatically on file changes
mix test.watch

# Check code style
mix dogma
```
